# Contributing to MiniLangCompiler

Thanks for your interest in `MiniLangCompiler`!

This project is a **personal reference implementation** and is **not open to external contributions**. It exists to demonstrate a specific architecture pattern and is not being developed as a collaborative open-source project.

## What You Can Do

- **Report bugs** — If something is broken, please [open an issue](https://github.com/ScottsSecondAct/MiniLangCompiler/issues). Include your OS, .NET version, and steps to reproduce.
- **Ask questions** — If something in the docs is unclear, an issue is fine for that too.
- **Fork it** — You're welcome to fork this repo and adapt it for your own purposes under the terms of the [MIT License](LICENSE).

## Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](CODE_OF_CONDUCT.md). Please read it before participating.
