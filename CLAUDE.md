# MiniLangCompiler

A compiler for a small custom language (MiniLang) that targets .NET assemblies. Written in C# (.NET 9.0).

## Project Goal

Wrap up the compiler by introducing a proper **AST (Abstract Syntax Tree)** as an intermediate representation between parsing and C# code generation. Currently, `MiniLangCompiler.cs` walks the ANTLR parse tree and emits C# strings directly in one pass — this should be split into two phases:

1. **Parse tree → AST**: A visitor that builds a typed AST (C# classes/records)
2. **AST → C# code**: A separate emitter that walks the AST and generates C# source

## Current Pipeline

```
.minilang source
  → ANTLR Lexer (MiniLangLexer.cs)
  → ANTLR Parser (MiniLangParser.cs)       produces parse tree
  → MiniLangCompiler.cs (visitor)          walks parse tree, emits C# strings directly
  → CSharpEmitter.cs                       wraps in Main(), compiles via Roslyn to .dll
```

## Target Pipeline (after AST work)

```
.minilang source
  → ANTLR Lexer/Parser                     produces parse tree
  → AstBuilder (new)                       walks parse tree, produces typed AST nodes
  → CSharpCodeGen (new/renamed)            walks AST, emits C# source string
  → CSharpEmitter.cs                       wraps in Main(), compiles via Roslyn to .dll
```

## Language Features (MiniLang)

- Types: `int`, `float`, `bool`, `string`
- Variable declaration: `int x = 5;` or `int x;`
- Assignment: `x = x + 1;`
- Arithmetic: `+`, `-`, `*`, `/`
- Comparison: `==`, `!=`, `<`, `>`, `<=`, `>=`
- If statements: `if (condition) { ... }`
- While loops: `while (condition) { ... }`
- Built-in: `print(expr)` → `Console.WriteLine(expr)`

## AST Design Notes

AST node types should mirror the grammar rules in `MiniLang.g4`. Use C# `record` types for immutability and concise syntax. Suggested structure:

```
AstNode (abstract base)
├── ProgramNode        — list of statements
├── Statements
│   ├── DeclStmt       — type, name, optional init expr
│   ├── AssignStmt     — name, value expr
│   ├── ExprStmt       — expression
│   ├── IfStmt         — condition expr, body block
│   └── WhileStmt      — condition expr, body block
├── BlockNode          — list of statements
└── Expressions
    ├── BinaryExpr     — left, op, right (covers arithmetic + comparison)
    ├── CallExpr       — function name, argument
    ├── IdentifierExpr — variable name
    ├── NumberExpr     — literal value
    └── ParensExpr     — inner expression
```

## Build & Run

